Please read the Chapter 6 of book to review the steps of how to make this package

This package will change the following system configurations:
	* Config timer server to time.asia.apple.com
	* Turn on timer server sync service
	* Set time zone to Aisia/Shanghai
	* Enable Apple Remote Desktop service and all admin with full control priviledge
	* set the four computer info items according to the computer configurations
	* Set Login window wallpaper to 

Install the following apps:
	* VLC Player.app
	* Google Chrome.app

Apply the following profiles:
	* User default desktop picture
	* Safari default homepage
