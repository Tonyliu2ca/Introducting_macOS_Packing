# ---------------
# 如何使用例子
# ---------------
1. 请将解包的目录改名为myPKGs，并将之复制到当前用户的桌面上，因为每个制作脚本的默认目录是$HOME/Desktop/myPKGs, 如果变更myPKGs目录位置或者文件名，请做相应变更，否则会出错
2. 请将EtreCheck.app复制到Chapter4/EtreCheck/root/Applications/Utilities目录中
3. 请将dockutil命令文件复制到Chapter5/docutil/root/bin目录中
4. 请先将Initialize, loginwallpaper和profiles中使用各自的make.sh生成组件包后，在打开Distribution2的Packages工程文件
